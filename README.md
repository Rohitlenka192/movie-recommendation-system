# Movie Recommendation System

A machine learning-based movie recommendation system built using Python and Scikit-learn.

## Features
- Content-based filtering
- Cosine similarity
- NLP preprocessing
- Movie recommendation engine

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy
- Jupyter Notebook

## Example
Input:
Avatar

Output:
- Beowulf
- Apollo 18
- Tears of the Sun
- The American
- Hanna
